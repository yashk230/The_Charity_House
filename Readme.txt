# The_Charity_House
This is a project totaly made in python and django.
It is a website that will host multiple NGO's and multiple USER's that would receive and make donations respectively 

To run this project follow this steps:
1. create a virtual environment named: "charity"
2. create a project named:"charity_house"
3. create a database in SQLite / MySQL workbench
4. Database name "charity"
5. Give the connections to the database
6. After setup run the project
